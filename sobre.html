Sobre la web

Oblitus est scientia nace como una biblioteca personal de red divulgativa: artículos largos, investigaciones, ensayos, documentos convertidos en publicaciones web y piezas interactivas.

Línea editorial

La página está pensada para publicar contenido con tono serio, estructura documental y vocación divulgativa. No busca funcionar como una red social ni como un blog rápido de actualidad, sino como un archivo ordenado de textos extensos sobre ciencia, tecnología, pensamiento crítico, historia, geopolítica del conocimiento e inteligencia artificial.

La prioridad visual es que el lector entre, localice la biblioteca y pueda abrir un artículo con el menor número posible de pasos. Las páginas explicativas quedan separadas para no cargar la portada con bloques institucionales innecesarios.

Sistema de publicación

Los documentos fuente se guardan en la carpeta articulos/. El proyecto mantiene una estructura estática para GitHub Pages: los documentos se transforman en HTML y el índice generado alimenta la portada.

  1. Se escribe o prepara un documento en Word, PDF, Markdown o HTML.
  2. El archivo se guarda dentro de articulos/.
  3. El generador construye el índice de publicaciones y los HTML internos.
  4. La portada muestra los artículos como mosaico ordenado por fecha.
  5. El lector interno abre el contenido como una publicación propia de la web.

Formatos previstos

Word / DOCX

Formato recomendado para artículos largos. Funciona mejor usando estilos reales: títulos, subtítulos, listas y tablas.

PDF

Útil para conservar documentos finales, aunque no siempre respeta la estructura al convertirlo a artículo web.

HTML interactivo

Formato indicado para publicaciones con animaciones, simulaciones o pequeñas microinterfaces JavaScript.

Markdown

Opción ligera para textos limpios, notas editoriales y documentos que no requieran maquetación compleja.
