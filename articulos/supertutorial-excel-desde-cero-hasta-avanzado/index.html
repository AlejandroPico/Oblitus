Capítulo 1: Introducción a Excel y Conceptos Básicos

Excel es una hoja de cálculo electrónica ampliamente utilizada para todo tipo de tareas, desde cálculos simples hasta análisis de grandes conjuntos de datos. Su potencial reside en la cuadrícula de celdas dispuestas en filas y columnas, donde podemos ingresar números, texto o fórmulas[1]. Estas celdas organizadas nos permiten sumar datos, ordenarlos, filtrarlos, crear tablas y elaborar gráficos visualmente atractivos[1]. En este capítulo sentaremos las bases: entenderemos qué es Excel, para qué sirve, y conoceremos su entorno de trabajo en la aplicación de escritorio de Office 365.

¿Qué es Excel?

Microsoft Excel es un programa de hoja de cálculo que forma parte de la suite Microsoft Office. Se utiliza en innumerables campos (negocios, ciencia de datos, finanzas, administración, etc.) para organizar información en forma de tabla y realizar cálculos o análisis con esa información. Por ejemplo, puedes usar Excel para llevar un registro de gastos personales, para generar reportes financieros, analizar resultados de ventas o incluso para tareas científicas. En resumen, Excel sirve para obtener información significativa a partir de datos[1], facilitando desde cálculos simples hasta el procesamiento de grandes volúmenes de datos.

Libros, hojas y celdas

Antes de comenzar a usar Excel, es importante dominar su terminología básica:

Libro de Excel: es el archivo en sí, con extensión .xlsx (o .xls en versiones antiguas). Un libro puede contener varias hojas de cálculo. Cuando abrimos Excel y creamos un archivo nuevo, estamos creando un libro en blanco[2].

Hoja de cálculo: es cada pestaña o página dentro del libro. Piensa en el libro como un cuaderno, y en las hojas de cálculo como las distintas hojas dentro de ese cuaderno. Cada hoja de cálculo está formada por una cuadrícula de celdas donde introduciremos los datos. Un libro puede tener una o muchas hojas (Excel las nombra por defecto "Hoja1", "Hoja2", etc., pero podemos renombrarlas).

Celda: es la intersección de una columna y una fila en la hoja. Cada pequeño rectángulo de la cuadrícula es una celda. Las celdas se identifican por una referencia compuesta de letra y número: la letra indica la columna y el número indica la fila (por ejemplo, A1 es la celda en columna A, fila 1). Las celdas son la unidad básica donde escribimos datos o fórmulas[3]. En cada celda solo puede haber un dato a la vez (ya sea un número, texto, fecha, fórmula, etc.).

Un concepto importante es que un libro de Excel es muy flexible: podemos agregar o eliminar hojas según necesitemos organizar los datos[2]. Por ejemplo, en un mismo archivo podrías tener una hoja para los ingresos, otra para gastos, otra para un resumen, etc., y todas conviven dentro del mismo libro.

La interfaz de Excel (versión de escritorio)

Al abrir Excel (usando la aplicación de escritorio de Office 365) nos encontraremos con la interfaz principal. Conocer sus partes nos ayuda a ubicarnos y aprovechar mejor la herramienta:

Cinta de opciones: es la barra de menús y botones que aparece en la parte superior de la ventana de Excel. Está organizada en pestañas (Inicio, Insertar, Fórmulas, Datos, Revisar, Vista, etc.). Cada pestaña agrupa comandos relacionados. Por ejemplo, la pestaña Inicio contiene herramientas básicas (copiar, pegar, formato de texto); la pestaña Insertar permite agregar gráficos, tablas, etc.

Barra de fórmulas: ubicada usualmente debajo de la cinta de opciones, muestra el contenido de la celda que tenemos seleccionada. Ahí podremos ver o editar valores y fórmulas de la celda activa. Es muy útil porque a veces una fórmula es larga y en la celda solo se ve el resultado; la barra de fórmulas nos deja ver la fórmula completa.

Columnas y filas: la hoja de cálculo está etiquetada con letras en las columnas (A, B, C,..., Z, luego AA, AB, etc.) en la parte superior, y con números en las filas (1, 2, 3, ...) a la izquierda. Estas letras y números sirven para identificar las celdas (como vimos, A1, B2, C10, etc.). Las columnas van de arriba hacia abajo y las filas de izquierda a derecha[4].

Celda activa: es la celda seleccionada actualmente en la hoja. Está resaltada con un borde más grueso. Podemos identificar cuál es mirando el rectángulo marcado, o viendo el nombre de la celda en un recuadro arriba a la izquierda (llamado "cuadro de nombre"), que indica por ejemplo "A1" si esa es la celda activa[5].

Otras partes: En la esquina inferior izquierda podemos ver pestañas que representan las hojas del libro (Hoja1, Hoja2, etc. – se pueden renombrar con doble clic). En la esquina inferior derecha suele haber controles de zoom y vista (para acercar/alejar la hoja, o cambiar entre vista normal, vista de salto de página, etc.). También hay una barra de estado en la parte inferior que muestra información útil, como por ejemplo la suma de valores seleccionados, el promedio, etc., de forma automática (si seleccionas varias celdas con números, Excel suele mostrar algunas estadísticas en esa barra de estado).

Nota: Dado que usarás la versión de escritorio de Excel, ten en cuenta que la apariencia puede variar ligeramente según la versión (Excel 2016, 2019, 2021 o 365). Office 365 suele actualizar la interfaz con pequeñas mejoras, pero en esencia las partes principales (cinta de opciones, barras, celdas) son las mismas. La versión web de Excel (Excel Online) es muy similar en funcionalidad básica, aunque la de escritorio suele tener algunas características avanzadas adicionales, especialmente útiles para macros u otras integraciones complejas, que aprenderemos más adelante.

Crear un nuevo libro de Excel

Cuando inicias Excel, normalmente aparece la pantalla de inicio donde puedes elegir entre plantillas o un Libro en blanco. Para empezar desde cero:

Abrir Excel: Inicia la aplicación Excel desde tu menú de inicio (Windows) o Launchpad (Mac).

Seleccionar "Libro en blanco": En la pantalla inicial, haz clic en la opción Libro en blanco (o "Blank workbook" si estuviera en inglés) para crear una nueva hoja de cálculo vacía[2]. Esto abrirá un nuevo libro llamado temporalmente Libro1 (Excel asigna nombres temporales hasta que guardes el archivo con un nombre propio).

Alternativamente, también puedes crear un nuevo libro desde la barra de menú: ve a la pestaña Archivo (en la cinta de opciones), luego selecciona Nuevo, y elige Libro en blanco[2]. En Office 365, puede que incluso tengas una barra de búsqueda donde escribir "nuevo libro" y te llevará a esa opción.

Plantillas: Excel ofrece plantillas prediseñadas para presupuestos, calendarios, facturas, etc. Si tu objetivo es aprender, lo mejor es comenzar con un libro en blanco para practicar. Más adelante podrías explorar las plantillas para ver ejemplos prácticos.

Introducir datos en las celdas

Una vez tengas tu nuevo libro abierto con una hoja en blanco (por defecto será "Hoja1"), puedes empezar a introducir datos. Para escribir en una celda, sigue estos pasos:

Selecciona una celda: haz clic con el ratón en la celda donde quieras introducir información. Por ejemplo, celda A1 (columna A, fila 1). Verás que la celda se resalta, indicando que está activa.

Escribe el valor: simplemente comienza a teclear. Puedes escribir números (por ejemplo, 100), texto (por ejemplo, Hola), fechas (01/07/2025), etc. Verás que lo que escribes aparece tanto en la celda como en la barra de fórmulas (mientras editas).

Confirma la entrada: tras escribir, pulsa la tecla Enter para confirmar. Al hacerlo, Excel registrará el valor en la celda y normalmente moverá la selección a la celda de abajo (por ejemplo, si estabas en A1 y pulsas Enter, pasarás a A2). Si en vez de Enter pulsas Tab, la celda activa se moverá a la derecha (útil para llenar una fila). También puedes simplemente hacer clic en otra celda para terminar de editar la actual.

¡Listo! Has ingresado tu primer dato en Excel. Repite el proceso en distintas celdas para practicar. Por ejemplo, prueba a escribir tu nombre en B1, un número en B2, etc.

Edición de celdas: Si necesitas corregir algo en una celda, tienes varias opciones: puedes volver a hacer clic en la celda y simplemente reescribir (lo nuevo reemplazará lo anterior), o hacer doble clic para editar dentro de la celda (el cursor se coloca dentro del texto de la celda). También puedes seleccionar la celda y editar en la barra de fórmulas (coloca el cursor donde necesites y cambia el contenido). Para confirmar los cambios, de nuevo Enter o click fuera.

Cancelar edición: Si mientras escribes en una celda te arrepientes, puedes pulsar Escape (Esc) y Excel cancelará la edición, dejando el valor anterior que tenía la celda.

Navegar y seleccionar celdas

Manejarse en la cuadrícula de Excel es una habilidad básica. Puedes usar tanto el ratón como el teclado:

Seleccionar con ratón: clic en una celda para seleccionarla. Para seleccionar varias celdas contiguas (un rango), haz clic en una y arrastra el cursor manteniendo pulsado el botón hasta cubrir las celdas deseadas, luego suelta. Las celdas seleccionadas se resaltarán. Esto es útil para, por ejemplo, resaltar un grupo de datos sobre los que quieras hacer algo (copiar, borrar, dar formato, etc.).

Seleccionar filas o columnas completas: haz clic en el encabezado de la fila (el número a la izquierda) para seleccionar toda esa fila. Haz clic en la letra de columna (en la parte superior) para seleccionar toda la columna. Si haces clic y arrastras sobre los encabezados, puedes seleccionar múltiples filas o columnas completas.

Uso del teclado: las flechas de dirección (←↑→↓) mueven la celda activa una posición en cada dirección. Puedes sostener la tecla Ctrl (o Cmd en Mac) junto con flechas para saltar al extremo de un bloque de datos (por ejemplo, Ctrl+→ salta al último valor del bloque de celdas contiguas hacia la derecha). También Page Up/Down desplaza la vista una pantalla arriba/abajo, Tab mueve una celda a la derecha (y Shift+Tab a la izquierda), Enter mueve una celda abajo (y Shift+Enter hacia arriba).

Ir a una celda específica: pulsa Ctrl+Ir o F5, se abrirá un cuadro "Ir a" donde puedes escribir una referencia (por ejemplo "Z1000") y saltarás directamente allí. Esto es útil en hojas muy grandes.

Practica moviéndote por la hoja: ve con las flechas, selecciona rangos arrastrando, prueba a seleccionar una fila entera, etc. La familiaridad con estos movimientos te hará más rápido usando Excel.

Guardar el libro de trabajo

Es fundamental guardar periódicamente tu trabajo en Excel para no perder datos. Al usar Office 365, tienes la posibilidad de guardar localmente en tu ordenador o en la nube (OneDrive) si has iniciado sesión con tu cuenta Microsoft. Pasos para guardar:

Guardar por primera vez ("Guardar como"): Ve a Archivo > Guardar como. Elige la ubicación: "Este equipo" para guardar en tu disco local, o OneDrive para guardar en la nube (lo cual permite acceder al archivo desde otros dispositivos y habilita autoguardado).

Nombre del archivo: Escribe un nombre descriptivo para tu libro, por ejemplo "Presupuesto2025.xlsx". Observa que Excel por defecto añadirá la extensión .xlsx. Elige la carpeta donde lo guardarás (por ejemplo, Documentos).

Guardar: Haz clic en el botón Guardar. A partir de ese momento, la barra de título de Excel mostrará el nombre que diste al archivo en lugar de "Libro1".

Una vez que el archivo tiene nombre y ubicación, cada vez que quieras guardar cambios basta con pulsar Ctrl+G (Cmd+G en Mac) o el ícono de guardar (un diskette) en la barra de herramientas de acceso rápido. Es buena práctica guardar con frecuencia para no perder avances.

Guardar automáticamente: Office 365 tiene una función de AutoGuardado cuando trabajas desde OneDrive, que guarda en segundo plano los cambios casi en tiempo real. Si estás guardando en tu PC local, no hay autoguardado continuo, así que acostúmbrate a guardar manualmente a menudo o activa la recuperación automática en caso de fallo (Excel suele recuperar archivos en caso de cierre inesperado, pero no dependas solo de eso).

Abrir un libro existente: Para abrir un archivo de Excel que ya tengas guardado, ve a Archivo > Abrir. Te mostrará una lista de archivos recientes y ubicaciones (OneDrive, Este equipo). Navega hasta el archivo .xlsx deseado y ábrelo. También puedes abrir un archivo directamente haciendo doble clic sobre él en tu explorador de archivos (lo cual lanzará Excel automáticamente).

Cerrar Excel: Cuando termines, puedes cerrar Excel desde Archivo > Cerrar o clic en la X (esquina superior derecha de la ventana). Si hay cambios no guardados, Excel te preguntará si deseas guardar antes de salir – elige "Sí" para no perder los últimos cambios.

Hasta aquí has aprendido a crear un libro, introducir datos básicos, moverte por la hoja y guardar tu trabajo. ¡Con esto ya tienes lo esencial para comenzar! En el siguiente capítulo empezaremos a explotar el poder de Excel con cálculos y fórmulas básicas.

Copiar, cortar y pegar

Excel proporciona las clásicas funciones de copiar, cortar y pegar que conoces de otros programas, pero adaptadas a las celdas:

Copiar (Ctrl+C): Sirve para duplicar el contenido de una celda o rango en otro lugar. Por ejemplo, si tienes un valor en A1 que quieres repetir en B1, selecciona A1 y copia (Ctrl+C, o clic derecho -> Copiar). Verás un borde punteado animado alrededor de la celda copiada.

Pegar (Ctrl+V): Selecciona la celda de destino (por ejemplo B1) y pega (Ctrl+V, o clic derecho -> Pegar). El contenido copiado aparecerá ahí. Importante: al pegar, por defecto Excel pega todo (el valor, la fórmula si la hay, y el formato). Más adelante veremos opciones para pegar solo valores o solo formatos.

Cortar (Ctrl+X): Es similar a copiar, pero indica que vas a mover el contenido original a otro lugar (quitándolo de su ubicación inicial). Cuando cortas, Excel también mostrará el borde punteado. Luego vas a la celda de destino y haces Pegar. El resultado es que los datos se trasladan. Por ejemplo, si cortas A1 y pegas en B1, A1 quedará vacía y B1 tendrá el contenido antes ubicado en A1.

Seleccionar rangos antes de pegar: Si copias un rango completo (varias celdas), cuando pegues, asegúrate de seleccionar la celda superior izquierda del área donde quieras pegar. Excel automáticamente ocupará el espacio necesario. Por ejemplo, si copias un bloque de 3 columnas por 4 filas, selecciona la celda que será la esquina superior izquierda del bloque de destino. Si el destino se sobrepone a celdas ya llenas, esas serán reemplazadas.

Atajos útiles: Además de Ctrl+C, Ctrl+V, también puedes usar el ratón: situar el cursor en el borde de una selección, arrastrar con tecla Ctrl presionada para copiar rápidamente arrastrando (observa que el cursor muestra un signo +). Sin Ctrl, arrastrar el borde mueve (corta/pega) el rango seleccionado.

Mover datos con arrastrar y soltar

Una forma rápida de mover datos en Excel es usando el ratón:

Selecciona la celda o rango que quieres mover.

Coloca el cursor sobre el borde de la selección hasta que cambie a un ícono de cuatro flechas (indicando "mover").

Haz clic y arrastra la selección hasta la nueva ubicación. Suelta el botón del ratón.

Esto es equivalente a cortar y pegar, pero en un solo gesto. Es muy práctico para reordenar datos en pequeñas distancias. Precaución: si arrastras y sueltas sobre celdas que tenían datos, los sobreescribirás. Excel no pide confirmación, así que asegúrate de no soltar accidentalmente en un lugar equivocado. Si te equivocas, recuerda que siempre puedes hacer Deshacer (Ctrl+Z) para revertir la acción.

También puedes copiar con arrastrar: arrastra con el botón derecho del ratón, y al soltar aparecerá un menú contextual preguntando si deseas mover o copiar. O bien, arrastra con el botón izquierdo pero manteniendo presionada la tecla Ctrl, como mencionamos.

Formato básico de las celdas

Formatear celdas consiste en cambiar la apariencia de los datos para mejorar la legibilidad o destacar cierta información. Excel ofrece muchas opciones de formato. Empecemos con las más comunes, disponibles en la pestaña Inicio de la cinta de opciones (grupo "Fuente" y "Alineación"):

Negrita, cursiva, subrayado: Funciona igual que en Word. Selecciona la celda o rango cuyos textos quieres resaltar y haz clic en Negrita (B) para poner el contenido en negrita. Cursiva (I) para itálica, Subrayado (U) para subrayar. Estos botones también tienen atajos: Ctrl+N para negrita, Ctrl+K para cursiva (K de Kursiva en español), Ctrl+S para subrayado.

Fuente y tamaño: Puedes cambiar el tipo de letra y el tamaño. Por ejemplo, para títulos podrías usar una fuente más grande. Selecciona la celda, en el menú desplegable de fuente elige otra (Arial, Calibri, Times, etc.) y ajusta el tamaño (por ejemplo 14, 16 puntos). Por defecto Excel suele usar Calibri 11 puntos.

Colores: Puedes cambiar el color de la fuente (el botón de la "A" subrayada de color) y el color de relleno de la celda (el ícono del tarro de pintura). Por ejemplo, si quieres resaltar una celda importante, puedes poner el texto en rojo o la celda con fondo amarillo. Úsalos con moderación para no sobrecargar la tabla visualmente.

Alineación: Por defecto, Excel alinea los números a la derecha de la celda y el texto a la izquierda. Puedes cambiar la alineación horizontal (izquierda, centro, derecha) con los botones correspondientes. También puedes alinear verticalmente (arriba, centro, abajo) si incrementaste la altura de la fila. Además, el botón "Ajustar texto" permite que si el texto es muy largo, se envuelva en varias líneas dentro de la misma celda (haciendo la fila más alta en consecuencia) en lugar de desbordarse hacia celdas adyacentes vacías.

Formato de números: Excel distingue el contenido de una celda entre valor numérico y texto. Un número puede visualizarse de diferentes maneras sin cambiar su valor. En la cinta Inicio, hay un desplegable para formato de número. Allí puedes elegir, por ejemplo, "Moneda" (para añadir el símbolo de moneda y dos decimales), "Porcentaje", "Fecha", "Hora", "Fracción", etc. Por ejemplo, si escribes 0.5 y luego aplicas formato de porcentaje, verás "50%". O si tienes 1200 y aplicas formato Moneda (asumiendo configuración regional de España) verás "1.200,00 €". Es útil para presentar correctamente datos numéricos. También puedes personalizar formatos (eso es más avanzado, con códigos de formato).

Bordes: Puedes dibujar bordes alrededor de celdas o rangos para delimitar secciones de tu tabla. El botón de Bordes te deja elegir borde inferior, superior, cuadriculado completo, etc. Por ejemplo, podrías poner bordes exteriores gruesos a un rango para destacarlo como tabla, y bordes internos tenues.

Tómate tiempo para experimentar: formatea algunos encabezados de columna en negrita y con fondo gris, datos importantes en otro color, etc. Un buen formato ayuda a entender mejor la hoja.

Ajustar ancho de columnas y alto de filas

A menudo, el texto o números que escribas pueden no caber en la celda con el ancho predeterminado de la columna. Si ves que el contenido se sale de la celda o aparecen ##### (almohadillas, que indican que un número no cabe en la celda), debes ajustar el tamaño:

Ajustar ancho automáticamente: La manera más rápida es situar el cursor en la línea divisoria entre dos letras de columnas en la cabecera (por ejemplo, entre la columna A y B, en la línea vertical que las separa). El cursor cambiará a una flecha doble horizontal. Haz doble clic allí, y Excel autoajustará el ancho de la columna A al contenido más ancho que haya en esa columna. Esto se llama Autoajustar.

Arrastrar para ajustar: También puedes hacer clic y arrastrar esa división de columna hacia la derecha o izquierda para manualmente establecer un ancho que te parezca adecuado. Aparecerá una etiqueta mostrando el ancho en píxeles o en caracteres.

Para filas (altura): Funciona igual pero en las divisiones entre números de fila a la izquierda. Aunque en filas generalmente no necesitas ajustar a menos que uses varias líneas de texto en una celda (con "Ajustar texto").

Formato > Autoajustar fila/columna: En el menú contextual (clic derecho en el encabezado) o en la cinta (pestaña Inicio > grupo Celdas > Formato) también hay opciones para ancho de columna y alto de fila exactos o autoajustados.

Haz pruebas escribiendo un texto largo en una celda, verás que se desborda mientras la celda de al lado esté vacía. Después ajústale el ancho de columna. Intenta también autoajustar todas las columnas seleccionando toda la hoja (clic en el recuadro en esquina superior izquierda de la cuadrícula, o Ctrl+E para seleccionar todo) y luego doble clic en cualquier división de columna: ajustará todas en un paso.

Sugerencia: Usa autoajustar para que tus datos sean visibles, pero si tienes muchas columnas es posible que quieras un ancho estándar para uniformidad. Encuentra un equilibrio entre visibilidad y un diseño limpio.

Con esto concluyen las operaciones básicas de edición y formato. Ya sabes cómo ingresar y modificar datos, copiarlos, moverlos y darles formato básico. Capítulo 4: Fórmulas y Funciones Básicas – Sumatorios, Promedios y Cálculos Simples

Entramos ahora a uno de los aspectos más poderosos de Excel: las fórmulas. Una fórmula en Excel es una expresión que realiza un cálculo utilizando los valores de ciertas celdas (u operaciones matemáticas definidas). Las fórmulas siempre comienzan con el signo = en una celda. Además de fórmulas personalizadas, Excel ofrece funciones predefinidas (SUMA, PROMEDIO, etc.) para cálculos comunes. En este capítulo aprenderemos a crear fórmulas básicas y a usar las funciones más sencillas, como sumas y promedios, obteniendo resultados automáticamente a partir de nuestros datos.

Cómo escribir una fórmula en Excel

Para introducir una fórmula, selecciona la celda donde quieres que aparezca el resultado del cálculo y comienza por escribir el signo igual =. Esto le indica a Excel que lo que vas a introducir es una fórmula y no texto plano. Después del =, escribe la expresión de tu cálculo.

Ejemplo sencillo: Si en la celda A1 has escrito el número 5 y en B1 el número 7, puedes ir a la celda C1 y escribir =A1 + B1. Al pulsar Enter, C1 mostrará el resultado de sumar las celdas A1 y B1, que en este caso es 12. Lo poderoso es que si luego cambias los valores en A1 o B1, la fórmula se recalculará automáticamente mostrando el nuevo resultado en C1.

Observa que en la barra de fórmulas se sigue viendo =A1+B1 cuando seleccionas C1, aunque en la celda se vea 12. Esto porque la celda almacena la fórmula, pero muestra su resultado.

Operaciones aritméticas básicas

Excel respeta el orden matemático de las operaciones (primero multiplicación/división, luego suma/resta, y paréntesis para forzar orden). Los operadores fundamentales:

Suma (+): =A1 + A2 suma los valores de A1 y A2.

Resta (-): =A1 - A2 resta el valor de A2 al de A1.

Multiplicación (*): =A1 * B1 multiplica A1 por B1. (Asterisco * es el signo de multiplicar).

División (/): =A1 / B1 divide A1 entre B1. Ten en cuenta que si B1 es 0, la fórmula dará un error #¡DIV/0!.

Exponenciación (^): (Por si lo necesitas) =A1 ^ 2 elevaría A1 al cuadrado, por ejemplo.

También puedes usar números "literales" en fórmulas: =A1 + 100 sumaría 100 al valor de A1; o =(A1 + A2) / 2 para sumar A1 y A2 y luego dividir entre 2.

Un caso práctico: imagina tener en A1 la cantidad de un producto (por ej., 8 unidades) y en B1 el precio unitario (por ej., 15). En C1 puedes calcular el costo total con =A1 * B1, así si A1 o B1 cambian, C1 siempre mostrará el producto de ambas.

Uso de funciones básicas: SUMA, PROMEDIO, etc.

Las funciones de Excel son fórmulas predefinidas que simplifican cálculos comunes. Todas las funciones tienen un nombre (por ejemplo SUMA) seguido de paréntesis () que encierran los argumentos (los valores o rangos sobre los que actúan). Veamos dos de las más usadas:

SUMA: Calcula la suma de un conjunto de celdas. Ejemplo: si tienes números en A1, A2, A3, puedes obtener la suma en A4 escribiendo =SUMA(A1:A3). El operador : significa un rango continuo desde A1 hasta A3. Al pulsar Enter, A4 mostrará la suma de esos tres valores. Puedes sumar celdas dispersas separándolas con punto y coma en los argumentos: =SUMA(A1; C1; E1) sumaría A1 + C1 + E1.

PROMEDIO: Calcula el promedio (media aritmética) de las celdas indicadas. Ejemplo: =PROMEDIO(B1:B10) calculará la media de los valores en el rango B1 a B10. Si la suma de esos valores es 100 y hay 10 celdas con números, el resultado sería 10 (o lo que corresponda).

Otras funciones útiles básicas incluyen: MAX (máximo valor de un rango), MIN (mínimo valor), CONTAR (cuenta cuántas celdas numéricas hay en el rango), CONTARA (cuenta celdas no vacías, sean número o texto). Por ejemplo, =MAX(C1:C50) te da el valor más alto encontrado entre C1 y C50.

Para insertar funciones fácilmente, Excel tiene un asistente: en la pestaña Fórmulas, a la izquierda, hay un botón "Insertar función" (ícono fx). O desde la pestaña Inicio, en el botón ∑ (Autosuma). Si seleccionas un rango contiguo de números y presionas el botón Autosuma ∑, Excel intenta crear la fórmula SUMA automáticamente para ese rango.

Nota: Los nombres de funciones en Excel en español se traducen (SUMA, SI, BUSCARV, etc., vs SUM, IF, VLOOKUP en inglés). Asegúrate de usar el nombre español si tu Excel está en español. Si alguna vez ves fórmulas en inglés, ten presente que tu versión las traducirá (por ejemplo, si escribes =SUM( en un Excel configurado en español, seguramente lo corrija a =SUMA().

Referencias de celdas en las fórmulas

Como vimos, puedes (y debes) usar referencias a celdas en las fórmulas en lugar de valores fijos, porque así los cálculos se actualizan cuando cambian los datos fuente. Esto convierte tu hoja en una especie de modelo dinámico. Un cambio en la entrada produce un cambio en el resultado automáticamente.

Cuando escribes una fórmula, en lugar de teclear "A1" puedes también simplemente hacer clic en la celda A1 y Excel insertará la referencia por ti. Si luego escribes + y haces clic en otra celda, la añadirá, y así. Esto ayuda a evitar errores de tipeo en referencias.

Además de referencias simples como A1, B2, etc., puedes referenciar celdas de otras hojas o incluso de otros libros, pero eso es más avanzado. Por ahora, nos centramos en referencias dentro de la misma hoja.

Recuerda: al crear fórmulas, asegúrate de que las referencias sean correctas. Si una fórmula apunta a una celda vacía, la tomará como 0 en cálculos numéricos. Si apunta a texto donde esperaba números, podría darte error de tipo. Excel tiene ayudas visuales: al editar una fórmula, resalta cada referencia con un color y dibuja un marco alrededor de la celda correspondiente, para que identifiques qué estás sumando, restando, etc.

Ejercicio sugerido: Escribe algunos números en una columna (por ejemplo en A1 a A5). En A6 usa =SUMA(A1:A5) para obtener la suma. Cambia alguno de los números de arriba y observa cómo A6 se actualiza al instante. Luego en B1 pon =PROMEDIO(A1:A5) para ver el promedio de esos mismos números. Juega cambiando datos, agregando quizás más números y ajustando el rango de la fórmula, etc., para afianzar cómo funcionan las referencias.

Capítulo 5: Referencias Relativas y Absolutas – Cómo Copiar Fórmulas Correctamente

Al crear fórmulas en Excel, es fundamental entender cómo funcionan las referencias de celda al copiar y pegar esas fórmulas. Por defecto, Excel usa referencias relativas, lo que significa que ajustará automáticamente las referencias cuando copiamos la fórmula a otra celda. Sin embargo, a veces queremos "fijar" alguna referencia para que no cambie; para ello usamos referencias absolutas. También existen referencias mixtas (parte relativa, parte absoluta). En este capítulo aprenderemos estos conceptos, que son esenciales para evitar errores al reutilizar fórmulas en múltiples celdas.

Temas y objetivos:

AsuntosObjetivos
Referencias relativas (comportamiento por defecto)Entender cómo cambian las referencias de celda cuando una fórmula se copia a otra posición.
Referencias absolutas (uso del símbolo $)Aprender a fijar filas o columnas en una fórmula usando $ para que no cambien al copiar.
Referencias mixtasConocer la existencia de combinaciones (columna fija, fila variable o viceversa) para usos avanzados.
Copiar fórmulas eficientementeSaber utilizar autorelleno o copiar-pegar fórmulas manteniendo la lógica correcta gracias a las referencias adecuadas.

Referencias relativas (el comportamiento estándar)

Una referencia relativa es la forma en que normalmente nos referimos a una celda (por ejemplo A1) en una fórmula. Se llama "relativa" porque si la fórmula se mueve, la referencia se interpreta relativa a la nueva posición.

Por ejemplo, supón que en la celda C1 tienes la fórmula =A1 + B1. Esta fórmula toma el valor dos columnas a la izquierda (col A) y lo suma con el valor una columna a la izquierda (col B) en la misma fila. Si copias esta fórmula de C1 hacia abajo a C2, la fórmula en C2 automáticamente se convierte en =A2 + B2. Observa que se ajustó la referencia de fila de 1 a 2, siguiendo la posición relativa (misma lógica: dos columnas a la izquierda y una a la izquierda, en la misma fila 2). Esto es muy útil porque te permite escribir una fórmula una vez y luego aplicarla a muchas filas sin tener que editar cada una manualmente[6][7].

Otro ejemplo: Si en D5 tienes =D4 * 2 (que multiplica el valor de arriba por 2), al copiar esa fórmula de D5 y pegarla en, digamos, E6, ¿qué ocurriría? Excel desplazará la referencia relativa a la nueva posición: E6 va a contener =E5 * 2. Se movió una columna a la derecha (D→E) y una fila abajo (4→5) siguiendo el movimiento de la fórmula.

En resumen, referencias relativas cambian en la misma magnitud que se mueve la fórmula: si bajaste una fila, las referencias de fila incrementan en 1; si la moviste 3 columnas a la derecha, las referencias de columna cambian 3 columnas hacia la derecha, etc.[6].

Referencias absolutas (usando $ para fijar filas o columnas)

A veces no queremos que una referencia cambie al copiar la fórmula. Por ejemplo, imagina que en una celda tienes un valor constante, como una tasa de impuesto, y quieres usarlo en fórmulas en varias partes de la hoja. Si usas una referencia normal, al copiar la fórmula podría desfasarse. Aquí es donde entran las referencias absolutas.

Se indica una referencia absoluta anteponiendo el símbolo $ antes de la columna y/o fila que quieres fijar. Por ejemplo:

$A$1 es una referencia absolutamente fija a la celda A1. No importa dónde copies la fórmula, seguirá apuntando exactamente a A1[7].

A$1 sería mixta: la fila 1 está fija (por el $ antes del 1), pero la columna A es relativa. Opuesto: $A1 fija la columna A pero deja relativa la fila.

Caso práctico: Supongamos que en la celda E1 tienes un porcentaje de IVA = 21%. En una columna A tienes precios netos y en columna B quieres calcular el precio con IVA. En B2 pondrías =A2 * $E$1 (multiplica el precio neto de A2 por la tasa en E1). Nota que hemos puesto $E$1 para que si copiamos esta fórmula hacia abajo en B3, B4..., siempre siga tomando la celda E1, donde está la tasa fija, sin moverse a E2, E3, etc.[7]. En cambio A2 sí la dejamos relativa, porque para cada fila queremos usar el precio de esa misma fila (A3 en B3, A4 en B4, etc.).

En B2: =A2 * $E$1

Copiada a B3: automáticamente sería =A3 * $E$1 (fíjate: A2 pasó a A3 por ser relativa; $E$1 quedó $E$1, igualita, por ser absoluta).

El símbolo $ se puede agregar fácilmente pulsando la tecla F4 cuando has seleccionado una referencia en la fórmula (al editar la fórmula). Cada pulsación de F4 cambia el tipo de referencia: relativa -> absoluta -> mixta (col fija) -> mixta (fila fija) -> de nuevo relativa, y así cíclicamente[8]. Es un atajo muy útil.

En resumen, usar $ en columna, fila o ambas nos permite mantener fija esa referencia al copiar[9]. Esto es crucial en muchos cálculos, especialmente cuando se usan celdas con valores constantes, referencias a tablas auxiliares, etc.

Ejemplo ilustrativo de relativa vs absoluta

Imagina la fórmula sencilla =A1 + A2 en la celda A3. Si la copias una fila hacia abajo a A4:

Como está, sin $, se convertirá en =A2 + A3 (todas las referencias bajaron una fila).

Si quisieras que siempre sumara los valores de A1 y A2 independientemente de dónde esté la fórmula, habrías que fijar las referencias: =$A$1 + $A$2. Copiada a A4, seguirá =$A$1 + $A$2.

Otro ejemplo: Tienes en columna C ventas de este año, en D ventas del año pasado, en E quieres la variación porcentual. En E2 pones =C2/D2 - 1 (porcentaje de cambio). Esa fórmula se copiará hacia abajo. Ambas referencias son relativas, lo cual es correcto aquí porque en cada fila quieres usar su propio C y D. No usarías $ porque no hay una celda fija común a todas las filas en este cálculo; cada fila es independiente.

Referencias mixtas (columna o fila fija)

Para completar: una referencia mixta es cuando solo una parte es absoluta. Ejemplo: $A1 (columna fija A, fila relativa) o A$1 (col relativa, fila fija 1). ¿Cuándo se usan? Un caso típico es cuando haces tablas de multiplicar o matrices de datos donde una dimensión se fija. Por ejemplo, supón que vas a construir una tabla donde las filas 2,3,4... multiplican un valor en la columna A por distintos porcentajes en las columnas B, C, D... Podrías necesitar fijar la columna de la base, pero dejar que la fila cambie, y viceversa. Son situaciones más avanzadas, pero es bueno saber que existen.

Copiar fórmulas con el controlador de relleno

Excel tiene una funcionalidad llamada autorrelleno: ese pequeño cuadrado negro en la esquina inferior derecha de la celda seleccionada. Si haces doble clic en ese cuadradito (cuando tienes una fórmula y hay datos contiguos a la izquierda), Excel rellenará automáticamente hacia abajo la fórmula hasta donde encuentre datos vecinos. Esto ahorra tener que arrastrar manualmente. Al hacerlo, las referencias se ajustarán automáticamente según sean relativas o absolutas.

También puedes arrastrar manualmente ese controlador de relleno hacia abajo o hacia la derecha para copiar la fórmula a varias celdas. Nuevamente, observarás cómo las referencias se ajustan.

Dominando las referencias relativas y absolutas evitarás errores como sumar rangos desplazados incorrectamente o apuntar a la celda equivocada. Si alguna vez copias una fórmula y obtienes un resultado inesperado, revisa las referencias: puede que necesites fijar algo con $. Es un concepto que al principio cuesta un poquito, pero es esencial en el manejo eficiente de Excel[9].

Ahora que sabemos armar fórmulas y controlarlas al copiar, pasemos a ampliar nuestro repertorio de funciones y herramientas de análisis en Excel.

Función SI (IF) – lógica condicional

La función SI te permite que Excel tome decisiones. Tiene la sintaxis:
=SI(condición_lógica, valor_si_verdadero, valor_si_falso).

Donde "condición_lógica" es una expresión que puede ser VERDADERO o FALSO (por ejemplo, A1 > 100, o B2="Sí"). Si la condición se cumple (es Verdadero), la función devuelve el "valor_si_verdadero"; si no se cumple (es Falso), devuelve el "valor_si_falso".

Ejemplo básico: Supón que en A2 tienes la nota de un estudiante, y quieres mostrar "Aprobado" si es >=5 o "Reprobado" si es <5. En B2 pondrías:
=SI(A2 >= 5, "Aprobado", "Reprobado").

Así, Excel evalúa A2 >= 5; si la nota es 7 (verdadero), B2 mostrará "Aprobado"; si la nota es 3 (falso), mostrará "Reprobado". Puedes arrastrar esa fórmula hacia abajo para evaluar muchas filas de notas.

La condición puede usar cualquier operador de comparación: =, >, <, >=, <=, <> (distinto de). También puede combinar condiciones con AND (Y) y OR (O), pero eso es más avanzado (requiere funciones Y, O anidadas dentro del SI).

SI anidados: ¿Qué pasa si hay más de dos casos? Por ejemplo, convertir notas numéricas a texto: 0-4 = "Insuficiente", 5-6 = "Suficiente", 7-8 = "Notable", 9-10 = "Excelente". Puedes anidar SIs:

=SI(A2<5, "Insuficiente",
SI(A2<7, "Suficiente",
SI(A2<9, "Notable",
"Excelente")))

Aquí hemos puesto otro SI como tercer argumento para manejar más casos. Excel evalúa secuencialmente hasta encontrar uno que cumpla. Nota: Los SIs anidados pueden ponerse pero se vuelven difíciles de leer cuando hay muchos; a veces conviene otras soluciones (como buscar en una tabla de rangos), pero es bueno conocer que se puede anidar.

Funciones de búsqueda: BUSCARV, BUSCARX (VLOOKUP, XLOOKUP)

Estas funciones permiten buscar un valor en una tabla y recuperar datos asociados. Son muy útiles para cruzar información. Explicaremos BUSCARV (muy popular) y mencionaremos XLOOKUP que es más nueva y poderosa (disponible en Office 365 y Excel 2021 en adelante).

BUSCARV (VLOOKUP en inglés) significa "búsqueda vertical". Sintaxis simplificada:
=BUSCARV(valor_buscado, rango_tabla, columna_devuelta, [ordenado]).

"valor_buscado": el valor que queremos encontrar en la primera columna del rango_tabla.

"rango_tabla": la matriz (tabla) donde buscar, que debe tener la columna clave a la izquierda.

"columna_devuelta": el número de columna (relativo al rango seleccionado) cuyo valor queremos obtener cuando se encuentre la fila coincidente.

"[ordenado]": opcional (VERDADERO/FALSO) para indicar búsqueda aproximada o exacta. Usualmente ponemos FALSO para búsqueda exacta.

Ejemplo: Tienes una tabla de productos donde la columna A es un código de producto y la B es el nombre. En otra parte, si alguien ingresa un código, quieres que automáticamente muestre el nombre. Si el código a buscar está en D2, podrías usar:
=BUSCARV(D2, $A$2:$B$100, 2, FALSO)

Esto busca el valor en D2 dentro de la primera columna del rango A2:B100. Cuando lo encuentre, devolverá la columna 2 de ese rango (o sea, la columna B, el nombre del producto). Ponemos FALSO para que busque coincidencia exacta (de lo contrario asume la lista ordenada para aproximación, lo cual no es deseado normalmente a menos que sepas lo que haces).

Limitaciones de BUSCARV: solo busca en la primera columna del rango y solo hacia la derecha. No puede buscar a la izquierda ni basado en columnas intermedias. También por defecto no distingue mayúsculas de minúsculas (no es case sensitive). Además, si hay duplicados, encuentra el primero.

XLOOKUP (BUSCARX): Es la versión moderna (disponible en Excel 365/2021). Simplifica mucho:
=BUSCARX(valor_buscado, rango_buscar_en, rango_a_devolver, [no_encontrado], [modo_busqueda]).

No requiere que la tabla esté ordenada ni que la columna de búsqueda sea la primera. Puedes especificar directamente dónde buscar y qué devolver. Por ejemplo, usando el mismo caso:
=BUSCARX(D2, $A$2:$A$100, $B$2:$B$100, "No encontrado")

Busca D2 en A2:A100, y devuelve el valor correspondiente de B2:B100. Si no lo encuentra, devolverá "No encontrado" (eso es el cuarto argumento opcional que hemos usado para manejar valores no encontrados de forma amigable). BUSCARX también puede buscar hacia la izquierda (podríamos haber puesto rango_buscar_en = B y rango_devolver = A si necesitáramos). Es más flexible e incluso puede buscar el valor más cercano mayor/menor, etc., pero lo básico es lo anterior. En Office 365 es muy recomendable usar BUSCARX sobre BUSCARV cuando esté disponible.

Coincidencia exacta vs aproximada: Con BUSCARV, poniendo FALSO forzamos exacta. Con BUSCARX, por defecto es exacta, pero se puede cambiar con el argumento de modo de búsqueda.

ÍNDICE y COINCIDIR: Son funciones más antiguas y versátiles para búsquedas complejas (COINCIDIR encuentra posición en un rango, ÍNDICE devuelve valor en posición). Las mencionamos pero quizá no profundizamos para no confundir; sepan que existen alternativas.

Otras funciones útiles de texto y fecha

Excel tiene una gran cantidad de funciones para manipular texto (strings) y fechas. Algunas fundamentales:

Funciones de texto:

CONCAT (o antiguamente CONCATENAR): Une textos. Ej: =CONCAT(A1, " ", B1) unirá el contenido de A1, un espacio en blanco, y B1. Útil para juntar nombres, crear códigos, etc. En Excel 2019+ también existe =CONCAT (similar) y =UNIRCADENAS (TEXTJOIN) para unir múltiples con separador.

IZQUIERDA, DERECHA, EXTRAE: Permiten tomar subcadenas. Ej: si A1 contiene "ABC123", =IZQUIERDA(A1,3) da "ABC" (los 3 caracteres de la izquierda). =DERECHA(A1,3) da "123". =EXTRAE(A1, 4, 3) empezando en pos 4, toma 3 caracteres (en este ejemplo también "123").

LARGO: =LARGO(A1) devuelve el número de caracteres en el texto de A1. Útil para validar longitudes.

MAYUSC, MINUSC: Pasan texto a MAYÚSCULAS o minúsculas.

ESPACIOS: =ESPACIOS(A1) quita espacios extra duplicados y espacios iniciales/finales. Útil al limpiar datos importados.

Funciones de fecha y hora:

HOY(): Retorna la fecha actual (la del día en que se calcule). No tiene argumentos, solo paréntesis vacíos.

AHORA(): Retorna fecha y hora actuales (como un número de fecha con decimales para la hora).

AÑO(fecha), MES(fecha), DIA(fecha): Extraen esas partes de una fecha dada. Si B1 tiene una fecha, =AÑO(B1) te da el año como número, MES(B1) el mes (1-12), etc.

DIASEM(fecha): Te dice qué día de la semana es (devolviendo un número 1-7, donde el 1 por defecto es domingo, pero se puede configurar).

FECHA(año, mes, día): Construye una fecha a partir de componentes numéricos.

DIAS(fecha_fin, fecha_inicio): Calcula la diferencia en días entre dos fechas.

Por ejemplo, podrías calcular la edad de alguien si tienes su fecha de nacimiento en C1: =INT((HOY() - C1) / 365) daría una aproximación (no exacta en todos los casos por años bisiestos, pero aproxima la edad en años). O usando funciones de fecha: hay métodos más exactos como AÑOSENTRE.

Convertir texto a número o viceversa: A veces los números vienen como texto. Funciones VALOR (convierte texto que representa número en número) o TEXTO (convierte número a texto con un formato especificado) pueden ser útiles.

Anidar funciones (combinar varias en una fórmula)

Excel permite que una función sea argumento de otra. Ya dimos un ejemplo con SI anidando OTROS SI. Otro ejemplo: imagina que tienes en una celda un nombre completo "Juan Pérez" y quieres extraer solo el apellido (todo lo que está después del espacio). No hay una función directa "APELLIDO", pero puedes combinar ENCONTRAR (para ubicar la posición del espacio) con DERECHA o EXTRAE. Por ejemplo, si A1 = "Juan Pérez":

=EXTRAE(A1, ENCONTRAR(" ", A1)+1, 999)

Aquí, ENCONTRAR(" ", A1) da la posición del primer espacio (5 en este caso). Le sumamos 1 para empezar desde después del espacio. Luego EXTRAE toma desde ese caracter hasta 999 (un número grande) caracteres, suficiente para capturar todo el resto (en Excel, si pides más caracteres de los que hay, simplemente devuelve hasta el final). Resultado: "Pérez". Este es un ejemplo de anidación de funciones (EXTRAE usando el resultado de ENCONTRAR). Es un poco técnico, pero muestra el poder de combinar.

Otro ejemplo de anidación: podríamos usar SI junto con funciones de texto. Imagina una lista en A donde algunos ítems terminan con "(OK)" y quieres en B poner "Correcto" si contienen "(OK)" o "Revisar" si no. Puedes usar la función HALLAR (similar a ENCONTRAR pero no distingue mayúsculas, o también ENCONTRAR, da igual) dentro de un SI:

=SI(ESERROR(ENCONTRAR("(OK)", A2)), "Revisar", "Correcto")

Aquí, ENCONTRAR("(OK)", A2) buscará la subcadena "(OK)" en A2. Si no la encuentra, produce un error #¡VALOR!. Usamos ESERROR() para capturar eso. Si es error (no lo encontró), entonces "Revisar"; si no es error (lo encontró), "Correcto". Vemos varios niveles: ENCONTRAR dentro de ESERROR, y todo eso dentro de SI.

La anidación permite cosas complejísimas, pero siempre es bueno mantener la fórmula entendible. A veces es mejor dividir el problema en columnas auxiliares ocultas que una sola mega fórmula. Sin embargo, para situaciones puntuales, combinar funciones ahorra pasos.

Consejo: Cuando anides funciones, asegúrate de que cada paréntesis tenga su cierre y de que las funciones están correctamente anidadas. Excel ayuda coloreando paréntesis y argumentos. Puedes también presionar Alt+Enter dentro de la barra de fórmulas para hacer saltos de línea y formatear la fórmula en múltiples líneas, facilitando la lectura (esto no afecta el cálculo, solo es visual en la barra de fórmulas).

Hemos cubierto bastante terreno: lógica con SI, búsquedas con BUSCARV/BUSCARX, manipulación de texto y fechas, y cómo combinar funciones. Con estas herramientas intermedias, ya se pueden resolver infinidad de problemas típicos en hojas de cálculo.

En los capítulos siguientes nos centraremos en herramientas de análisis de datos más globales: gráficos para visualizar información, tablas dinámicas para resumir grandes tablas, y también exploraremos la automatización con macros y la integración de Excel con código, que es uno de tus intereses particulares.

Capítulo 7: Análisis de Datos – Ordenar, Filtrar y Formato Condicional

Excel no solo sirve para cálculos, también es potente para analizar y organizar datos. En este capítulo nos enfocaremos en tres herramientas básicas de análisis: la capacidad de ordenar datos (de A a Z, de mayor a menor, etc.), la función de filtro automático para mostrar solo registros que cumplen ciertos criterios, y el formato condicional, que resalta automáticamente celdas que cumplen condiciones (por ejemplo, valores superiores a cierto umbral, duplicados, etc.) para identificar patrones o anomalías visualmente[10].

Temas y objetivos:

AsuntosObjetivos
Ordenar datos (alfabético, numérico)Aprender a ordenar tablas completas según una o varias columnas, de forma ascendente o descendente.
Filtros automáticosSaber filtrar datos en tablas para mostrar solo la información relevante según criterios (texto, números, fechas).
Formato condicional básicoAplicar reglas de formato condicional para resaltar información (por ejemplo, valores máximos/mínimos, duplicados, superiores al promedio, etc.)[10].
Limpiar datos y eliminar duplicadosConocer opciones para quitar duplicados y hacer pequeñas transformaciones que ayudan a preparar datos para análisis.

Ordenar datos

Cuando tienes una tabla (varias filas de registros y columnas de campos), a menudo querrás ordenarla. Por ejemplo, ordenar una lista de clientes por apellido alfabéticamente, o una tabla de ventas desde la venta más alta hasta la más baja.

Cómo ordenar:

Clic en cualquier celda dentro de la columna por la que quieras ordenar.

En la pestaña Datos de la cinta, encontrarás los botones Ordenar Ascendente (AZ) y Ordenar Descendente (ZA). Si los pulsas, Excel intentará ordenar toda la tabla usando esa columna como clave.

Es importante que tu tabla de datos esté contigua (sin filas o columnas vacías en medio) para que Excel infiera correctamente el rango a ordenar. Excel suele preguntar "¿expandir la selección?" si detecta más datos contiguos, dile que sí para que ordene todas las columnas juntas. De lo contrario, se desalinearían los datos (¡peligroso!).

Si necesitas ordenar por más de un criterio, usa el botón Ordenar (avanzado) que abre un cuadro de diálogo. Ahí puedes añadir niveles: por ejemplo, primero ordenar por "Departamento" (alfabético), y dentro de cada departamento, ordenar por "Fecha" (más reciente primero).

Orden ascendente con texto = A a Z; con números = menor a mayor; con fechas = más antiguas a más recientes. Orden descendente es lo opuesto. También hay orden personalizado (por ejemplo, listas personalizadas como ordenar los meses "Ene, Feb, Mar..." en vez de alfabético) en el cuadro de diálogo avanzado.

Excel mantiene cada fila unida al ordenar. Así, toda la información de una fila "viaja junta" para que no se mezclen los campos de registros distintos.

Filtrar datos

Los filtros nos permiten mostrar temporalmente solo las filas que cumplen cierta condición, ocultando las demás (no se borran, solo se ocultan mientras el filtro está activo). Esto es muy útil para explorar grandes tablas.

Activar filtros: Haz clic en cualquier celda del encabezado de tu tabla (asegúrate de tener una fila de encabezados clara, p. ej., títulos de columnas en la primera fila). En la pestaña Datos, pulsa el botón Filtro (icono de embudo). Verás que en cada celda de la fila de encabezados aparecen pequeñas flechas desplegables.

Usar los filtros: Haz clic en la flechita de la columna por la que quieras filtrar. Aparece un menú con opciones: - Un cuadro de búsqueda (útil si hay muchos valores distintos). - Una lista de valores únicos existentes en esa columna con casillas para marcar/desmarcar. - Opciones de Filtro de texto o Filtro de número (dependiendo del tipo de datos) que permiten condiciones más complejas: por ejemplo "contiene...", "empieza por...", "es mayor que...", "entre ... y ...", etc.

Ejemplo: Tienes una lista de ventas con una columna "Ciudad". Si solo quieres ver ventas de "Barcelona", puedes desmarcar todo y marcar solo "Barcelona" en el filtro de esa columna. La tabla se reducirá a mostrar solo filas de Barcelona. Puedes aplicar filtros en múltiples columnas simultáneamente (cada uno va refinando más). Por ejemplo, luego en la columna "Monto" podrías poner un filtro de número "Mayor que 1000" para ver ventas >1000 € en Barcelona.

Mientras un filtro esté activo, la flecha del encabezado cambia a un icono de embudo. Puedes borrar un filtro de una columna abriendo de nuevo el menú y eligiendo "Borrar filtro de ...".

Sugerencia: Cuando hay filtros activos, Excel te indica cuántas de cuántas entradas se muestran en la barra de estado. Asegúrate de quitar los filtros cuando ya no los necesites, para no olvidar que hay datos ocultos. Para quitarlos todos de golpe, puedes volver a Datos > Filtro (botón) para desactivar la función (esto restablece la vista completa).

Formato condicional para resaltar información

El formato condicional aplica automáticamente un formato (colores de relleno, color de fuente, iconos, barras, etc.) a celdas según su valor, facilitando visualizar tendencias o excepciones[10].

Para usarlo: 1. Selecciona el rango de celdas al cual quieres aplicar una regla de formato condicional (por ejemplo una columna de números, o todo un bloque). 2. Ve a la pestaña Inicio, grupo "Estilos", y haz clic en Formato Condicional. 3. Encontrarás varias opciones predefinidas: - Resaltar reglas de celdas: para marcar celdas mayores que un valor, menores que, entre dos valores, iguales a un texto específico, fechas, o que contengan duplicados. Ej: "Resaltar celdas > 1000" y elegir un color rojo claro de relleno. - Reglas superiores e inferiores: marca automáticamente el top 10 elementos, el top 10%, los inferiores, las celdas por encima del promedio, etc. - Barras de datos: dibuja barras horizontales dentro de la celda proporcionales al valor (como una mini-gráfica de barra). - Escalas de color: colorea las celdas con un degradado según su valor relativo (por ejemplo del rojo para valores bajos al verde para altos). - Conjuntos de iconos: pone iconitos (flechas, semáforos, estrellas) según tramos de valores.

Por ejemplo, supón que tienes una columna de "Días de demora" en pagos. Puedes aplicar un formato condicional: celdas > 30 días en rojo, 15-30 en amarillo, <15 en verde. Excel te permitirá hacerlo con dos reglas: "Mayor que 30 = relleno rojo", "Entre 15 y 30 = relleno amarillo", y por omisión podrías dejar las <15 sin formato o con otra regla "Menor que 15 = relleno verde".

También es muy útil resaltar duplicados: selecciona la columna que quieras revisar, Formato condicional > Resaltar reglas > "Valores duplicados...". Escoge un formato, y Excel marcará aquellas celdas cuyo valor aparezca al menos dos veces en esa selección. Así puedes detectar entradas repetidas en listas donde debería haber valores únicos (como códigos, DNI, etc.).

Las reglas de formato condicional se pueden administrar y editar: en Formato condicional > Administrar reglas puedes ver todas las reglas creadas, el orden, los rangos a los que aplican, y modificarlas o eliminarlas.

Nota avanzada: Formato condicional también permite usar fórmulas como criterio de formato, dándote una flexibilidad enorme (por ejemplo, resaltar toda la fila si cierta columna dice "Urgente"), pero eso lo dejamos como mención ya que implica manejar referencias absolutas/aplicables a rango en el contexto de reglas.

Eliminar duplicados y otras herramientas de datos

En la pestaña Datos, además de Ordenar y Filtrar, hay otras herramientas: - Quitar duplicados: Si seleccionas una tabla (o solo una columna) y pulsas "Quitar duplicados", Excel eliminará filas duplicadas basándose en las columnas que indiques. Te pedirá confirmar qué columnas considerar para determinar duplicados. Úsalo con cuidado, ya que borrará datos irreversiblemente (mejor haz copia antes o asegúrate de entender qué va a borrar). Esto es útil para limpiar listas (por ejemplo, obtener lista única de clientes a partir de un listado donde aparecen repetidos). - Validación de datos: Permite restringir qué se puede introducir en una celda (por ejemplo, solo números entre X e Y, o elegir de una lista desplegable). Esto es más para entrada de datos y evitar errores de usuario. No es análisis, pero es bueno saber que existe para cuando diseñes plantillas. - Texto en columnas: Sirve para separar contenido de una celda en varias columnas (por un delimitador, coma, punto y coma, etc.). Útil si importaste datos donde, por ejemplo, nombre y apellido están juntos y los quieres separar. - Combinar (Consolidar): hay herramientas para consolidar datos de varias hojas, pero esas son menos usadas en nivel básico-intermedio.

Tras aplicar ordenamientos, filtros y formato condicional, estarás en capacidad de entresacar información clave de conjuntos de datos. Por ejemplo, filtrar transacciones de cierto mes, resaltar las mayores ventas, ordenar por puntuación para ver top N resultados, etc. Estas técnicas preparan el terreno para el próximo capítulo, donde abordaremos gráficos para visualización y tablas dinámicas para resumir datos agregados.

Capítulo 8: Visualización de Datos – Gráficos y Diagramas

Una imagen vale más que mil filas de datos. Excel ofrece herramientas de gráficos que permiten convertir tablas de números en visualizaciones que facilitan la interpretación de la información. En este capítulo aprenderemos a crear gráficos básicos (columnas, líneas, sectores) y a personalizarlos. También veremos gráficos más avanzados como gráficos dinámicos o combinados. Al final, mencionaremos brevemente la existencia de herramientas de visualización avanzadas (como Power BI o complementos) pero nos centraremos en lo que se puede hacer directamente en Excel.

Temas y objetivos:

AsuntosObjetivos
Creación de gráficos básicosAprender a insertar gráficos comunes (barras/columnas, líneas, circular) a partir de datos seleccionados[11].
Personalizar elementos del gráficoSaber editar títulos, etiquetas, leyendas, colores y estilos de los gráficos para hacerlos claros y profesionales.
Gráficos dinámicos (vinculados a tablas dinámicas)Introducir el concepto de gráfico dinámico, que se actualiza con las segmentaciones de datos de una tabla dinámica.
Tipos especiales de gráficosConocer otros tipos: gráficos combinados (ej. columnas + línea), gráficos de dispersión (xy), histogramas, mapas (en las últimas versiones), etc., y cuándo podrían usarse.

Crear un gráfico básico desde datos

Supongamos que tienes una tabla sencilla: en la columna A los meses (Enero, Febrero, ...), y en la columna B las ventas de cada mes. Queremos visualizar la tendencia de ventas en el año.

Pasos para insertar un gráfico:

Selecciona el rango de datos que quieres graficar, incluyendo normalmente las etiquetas (en este caso, A1:A13 con los nombres de meses y B1:B13 con las ventas). Asegúrate que las etiquetas estén en la primera fila o primera columna para que Excel las use inteligentemente.

Ve a la pestaña Insertar en la cinta de opciones. Verás un grupo de botones de Gráficos: Columnas o Barras, Líneas, Circular (pie chart), Áreas, Dispersión, etc.

Haz clic en el tipo de gráfico deseado. Por ejemplo, elige Insertar gráfico de columnas -> Columna agrupada 2D (la típica). Excel insertará un gráfico en la hoja (generalmente como objeto flotante). Cada columna representará las ventas de un mes.

Si tus datos estaban bien seleccionados, Excel automáticamente asignará los ejes: el eje horizontal (categorías) serán los meses, el eje vertical (valores) la escala numérica de ventas. La serie (o series) de datos se tomó de la columna B.

En Excel 365 también existe la opción Gráficos recomendados (Insertar > Gráficos recomendados) que analiza tus datos y sugiere qué gráfico podría representarlos mejor. Es útil cuando no estás seguro qué elegir.

Personalización de gráficos

Una vez creado el gráfico, debería estar seleccionado (si no, haz clic sobre él). Verás posiblemente en la cinta una pestaña contextual "Diseño de gráfico" y "Formato" (o en versiones más recientes aparece un signo más, un pincel y un embudo junto al gráfico al seleccionarlo):

Título del gráfico: Por defecto podría decir "Título del gráfico". Haz clic sobre ese texto en el gráfico para editarlo. Pon un título descriptivo, por ejemplo "Ventas Mensuales 2025".

Ejes: Excel coloca automáticamente ejes con escala adecuada. Puedes hacer clic en los números del eje vertical y formatearlos (clic derecho > Formato de eje) para ajustar cosas como la unidad, agregar signo de moneda, etc. También puedes cambiar la orientación de etiquetas en el eje horizontal si se ven muy juntas (por ejemplo, en formato de eje horizontal puedes angulas las etiquetas 45°).

Leyenda: En un gráfico con una sola serie, la leyenda puede no ser necesaria (solo repetiría "Ventas" quizá). Puedes seleccionarla y suprimirla. En gráficos con varias series (por ejemplo, si tu tabla tuviera columnas Ventas y Gastos, y graficas ambas, la leyenda con colores para cada serie sí es útil).

Etiquetas de datos: Son los numeritos sobre cada columna (o punto) indicando su valor exacto. Puedes activarlas haciendo clic en el signo + que aparece al lado del gráfico (en versiones nuevas) y marcando "Etiquetas de datos". O desde diseño de gráfico > Agregar elemento de gráfico > Etiquetas de datos.

Colores y estilos: En la pestaña contextual Diseño de gráfico suele haber galerías de Estilos de gráfico predefinidos (combinan colores, efectos). También hay la pestaña Formato para cambiar colores manualmente de cada elemento. Por ejemplo, para cambiar el color de las barras, haz clic en una barra (todas se seleccionan si es una serie única) luego botón derecho > Formato de serie de datos > Relleno, y escoge un color. O usa el pincel (Estilos) que aparece flotando para elegir un tema de color.

Tamaño y posición: Puedes arrastrar las esquinas del gráfico para redimensionarlo. También arrastrarlo a otra posición en la hoja. Si prefieres un gráfico en hoja separada (una hoja de gráfico), puedes hacer clic derecho > Mover gráfico... > "Nueva hoja". Eso lo pondrá en una hoja dedicada, a veces útil para impresiones o presentaciones.

Excel permite muchos refinamientos: líneas de cuadrícula (puedes quitar o añadir), rótulos de ejes (para poner "Meses" o "Ventas en €"), etc. Explora el menú "Agregar elemento de gráfico" en la pestaña Diseño o el signo + como dije.

Gráficos de distintos tipos y usos

Columnas/Barras: Ideales para comparar magnitudes entre categorías. Columnas se usan típicamente para categorías discretas no demasiadas (ej: meses del año). Barras es lo mismo pero horizontal, útil cuando las etiquetas son largas o hay muchas categorías.

Líneas: Buenas para visualizar tendencias en el tiempo (serie temporal). Por ejemplo ventas mensuales se pueden graficar también con una línea uniendo puntos para resaltar la evolución continua.

Circular (Pie chart): Muestra proporciones de un total. Se usa cuando tienes, por ejemplo, distribución porcentual de un presupuesto en diferentes partidas. Se selecciona una sola serie de datos (no varias columnas, sino, por ejemplo, las ventas por región como porción del total). Ten en cuenta que un gráfico circular no debe tener demasiados segmentos (mejor menos de 6 u 8) y todos deben sumar 100%. Excel permite resaltar un segmento sacándolo ligeramente (efecto explosion).

Dispersión (XY): Se usa para conjuntos de pares de datos numéricos, para ver correlación. Por ejemplo, altura vs peso de individuos. No tiene categorías predefinidas, cada punto se coloca según su valor en eje X y eje Y. Útil para análisis científicos, de laboratorio, etc.

Áreas: Similar a líneas pero el área bajo la línea va coloreada. Útil para acumular valores o ver contribuciones.

Histogramas / Pareto: Excel 2016+ incorporó histogramas automáticos que agrupan valores en rangos. Útil para distribuciones de frecuencia.

Gráfico combinado: Combina dos tipos en uno, por ejemplo, columnas para ventas y una línea superpuesta para porcentaje de crecimiento. Esto se hace insertando un gráfico y luego cambiando el tipo de una serie (clic derecho en la serie > Cambiar tipo de gráfico de serie...). Puedes incluso asignar un eje secundario a una serie si tienen escalas muy diferentes.

Minigráficos: Son gráficos muy pequeños que caben en una celda (línea, columna o ganancia/pérdida). Insertar > Minigráfico. Sirven como indicadores compactos al lado de datos, por ejemplo, línea de tendencia de cada fila en una tabla. Estos no tienen muchos elementos extras, son más simples.

Excel facilita también el uso de segmentación de datos y gráficos dinámicos cuando usas tablas dinámicas (lo veremos enseguida). Además, con Office 365 hay integración incluso con Power BI y recomendaciones via IA (Ideas) que sugieren gráficos y análisis automáticamente.

Gráficos dinámicos vinculados a tablas dinámicas

Si en el siguiente capítulo creamos una Tabla dinámica (PivotTable) para resumir datos, podemos agregarle un Gráfico dinámico (PivotChart). La ventaja es que cuando filtres o actualices la tabla dinámica, el gráfico refleja esos cambios automáticamente.

Para crear uno, simplemente haz clic en la tabla dinámica, ve a la pestaña Analizar (o contextual de tabla dinámica) > Insertar gráfico. O inserta un gráfico normal pero dentro de las opciones selecciona la tabla dinámica como origen de datos.

También existen segmentadores (slicers) que son botones interactivos para filtrar tablas dinámicas (y por ende sus gráficos) de forma visual. Por ejemplo, un segmentador de "Año" con botones 2024, 2025, etc. Si pulsas 2025, la tabla dinámica y gráfico dinámico muestran solo datos de ese año.

Otros consejos de visualización

No sobrecargar: Elige el tipo de gráfico adecuado y no le pongas demasiada información. A veces es mejor usar dos gráficos simples que uno demasiado complejo.

Añadir contexto: Usa títulos claros, ejes rotulados. Si mandas a alguien un gráfico "Ventas Q1-Q4" asegúrate que los ejes o colores se entienden (e.g., euros en miles en eje Y, etc.).

Actualizar datos: Si los datos fuente cambian o se expanden (más filas), puede que necesites actualizar el origen del gráfico. Si tu rango era A1:B13 y añades datos en A14-B14, tendrías que arrastrar el borde azul que indica datos seleccionados (al seleccionar el gráfico se muestran) para incluir esa fila, o bien ajustar el rango en el menú Seleccionar datos.

Herramientas externas: Menciono que existen complementos como Power Map (ahora llamado 3D Maps) para datos geográficos en 3D, o la posibilidad de usar Python en Excel (que veremos luego) para generar visualizaciones avanzadas con librerías como matplotlib o seaborn dentro de Excel. Pero para la mayoría de usos, los gráficos nativos de Excel son suficientes.

Ahora ya puedes transformar tus hojas de cálculo en gráficas comprensibles. A continuación, pasaremos a una herramienta muy poderosa de Excel para análisis: las Tablas dinámicas, con las cuales podrás resumir y agregar datos sin necesidad de escribir fórmulas complejas.

Capítulo 9: Tablas Dinámicas – Resumiendo Grandes Volúmenes de Datos

Las tablas dinámicas (Pivot Tables) son una de las funcionalidades más potentes de Excel para el análisis de datos. Permiten tomar una gran tabla plana de datos (por ejemplo, todas las ventas detalladas: fecha, producto, región, importe, etc.) y rápidamente generar resúmenes agregados: totales por producto, totales por región y mes, conteos, promedios, etc., con la facilidad de arrastrar campos. Además, son interactivas: puedes reorganizar los campos (pivotar) para ver la información desde diferentes perspectivas[11]. En este capítulo aprenderemos a crear una tabla dinámica básica, configurar sus filas, columnas y valores, aplicar filtros o segmentaciones, y actualizarla cuando cambien los datos origen.

Temas y objetivos:

AsuntosObjetivos
Crear una tabla dinámica a partir de datosAprender el proceso de inserción de tabla dinámica desde una tabla de datos existente (selección de rango o tabla, elección de ubicación).
Estructura de la tabla dinámica (filas, columnas, valores)Comprender cómo arrastrar campos a las áreas de filas, columnas, valores (y filtro) para dar forma al resumen deseado[11].
Funciones de valor (suma, cuenta, promedio)Saber cambiar la operación de resumen (suma, cuenta, etc.) según el tipo de análisis necesario.
Filtrar y segmentar la tabla dinámicaUtilizar filtros de informe o segmentadores para profundizar en datos específicos, así como actualizar la tabla dinámica cuando los datos fuente cambian.

Preparar los datos para la tabla dinámica

Para crear una tabla dinámica efectiva, tus datos fuente deben estar estructurados de manera tabular, con: - Encabezados claros en la primera fila para cada columna (sin celdas combinadas, sin subtítulos dobles). - Datos contiguos (sin filas o columnas totalmente vacías en medio). - De preferencia, convertir el rango de datos en una Tabla de Excel (Selecciona cualquiera de sus celdas y presiona Ctrl+T, o menú Insertar > Tabla, marcando "La tabla tiene encabezados"). Esto no es obligatorio pero ayuda porque la tabla se expande automáticamente al agregar datos y la tabla dinámica puede reconocer el nombre de la tabla.

Supongamos que tenemos un conjunto de datos de ventas con columnas: Fecha, Vendedor, Región, Producto, Cantidad, Importe. Y hay cientos o miles de filas (cada fila una transacción).

Crear la tabla dinámica

Clic en cualquier celda dentro del rango de datos (o tabla).

Ve a Insertar > Tabla Dinámica. Excel abrirá un diálogo:

Seleccionar los datos: Debe detectar automáticamente el rango completo. Si usaste una Tabla, pondrá el nombre de la tabla (ej. Tabla1).

Elegir dónde colocarla: Por lo general elige "Nueva hoja de cálculo" (recomendado para que tu tabla dinámica aparezca en una hoja nueva). O podrías situarla en una hoja existente indicando la celda.

Pulsa Aceptar. Excel creará una nueva hoja con un lienzo en blanco de tabla dinámica y a la derecha aparecerá el Panel de campos de tabla dinámica.

El panel tiene la lista de campos (cada columna de tus datos se convierte en un "campo") y cuatro áreas: Filtros, Columnas, Filas, Valores.

Diseñar la tabla dinámica arrastrando campos

Ahora decides qué quieres analizar. Por ejemplo: Total de Importe por Región y por Producto.

Arrastra el campo Región al área de Filas. Verás en la tabla dinámica que aparecen las distintas regiones como filas (una por valor único de región).

Arrastra el campo Producto al área de Columnas. Aparecen en la tabla dinámica como encabezados de columna cada producto.

Arrastra el campo Importe al área de Valores. Por defecto, Excel pondrá "Suma de Importe" y llenará la tabla con los totales de importe para cada combinación de región y producto[11]. También pondrá un Gran Total por filas y columnas (sumando todas las regiones y todos los productos, respectivamente).

¡Y listo! Has creado un resumen donde cada celda interior es la suma de ventas de ese producto en esa región. Puedes interpretar filas, columnas, etc. Excel calculó todo por ti sin que tú escribas una sola fórmula.

Cambiar el tipo de cálculo: Si hubieras arrastrado un campo no numérico a Valores, Excel lo pondría como "Recuento" (ya que no puede sumar texto). También puedes cambiar la función. Por ejemplo, si arrastras Cantidad a valores y quisieras ver el promedio en lugar de la suma, haz clic en la flechita del campo en el área Valores > Configuración de campo de valor > Elige "Promedio". Puedes hacer conteos, máximos, mínimos, etc. La mayoría de veces Suma o Cuenta son los más usados.

Más de un campo en filas o columnas: Puedes anidar niveles. Por ejemplo, podrías poner Vendedor en Filas debajo de Región. Entonces la tabla dinámica agrupará por región, y dentro de cada región desglosará por vendedor. Podrás expandir/contraer con +/-. Lo mismo con columnas, podrías poner Año y Mes (si tuvieses fechas) en columnas para ver una matriz Año x Mes x Región etc.

Filtros de informe: Si arrastras un campo al área Filtros, aparecerá ese campo como un desplegable encima de la tabla dinámica. Eso te permite filtrar toda la tabla dinámica por ese campo. Por ejemplo, podrías poner "Año" en Filtros, y seleccionar 2025 para ver la tabla solo con datos de ese año. Si quisieras permitir selección múltiple (varios años), marca la opción en el desplegable de filtro "Seleccionar varios".

Actualizar y manejar la tabla dinámica

Actualizar datos: Si modificas o agregas datos en la tabla original, la tabla dinámica no se actualiza automáticamente (a menos que uses modelos de datos o ciertas conexiones, pero en general hay que refrescar). Para actualizar, haz clic derecho sobre la tabla dinámica > Actualizar. O ve a la pestaña contextual "Analizar/ Opciones de tabla dinámica" > botón Actualizar.

Cambiar campos: Puedes arrastrar campos fuera para quitarlos, o intercambiarlos de área. Juega con diferentes combinaciones. Por ejemplo, arrastra Producto a Filas y Región a Columnas si quieres pivotar la vista. Todo es muy dinámico, de ahí su nombre.

Diseño y formato: En la pestaña Diseño (contextual cuando estás en la tabla dinámica) puedes elegir si mostrar subtotales, totales generales, diseño compacto vs tabular, estilos de tabla dinámica predefinidos (colores alternos de fila, etc.).

Ordenar dentro de la tabla dinámica: Puedes ordenar, por ejemplo, las regiones por el total de ventas. Selecciona la columna de totales de región y ordena descendentemente. Excel te preguntará si deseas aplicar ese orden dentro de cada campo. Generalmente "Sí".

Filtrar con segmentadores: Una forma más visual que el filtro de informe es insertar Segmentaciones. Ve a Tabla Dinámica > Analizar > Insertar Segmentación de datos. Marca por ejemplo "Región". Esto agrega un panel con botones para cada región; pulsando en ellos filtras la tabla dinámica. Puedes seleccionar varios manteniendo Ctrl. Los segmentadores son prácticos para dashboards interactivos. Lo mismo con Escala de tiempo (slicer especial para fechas).

Actualizar rango de origen: Si tu tabla de datos original creció más allá del rango inicial y no era una "Tabla" de Excel, puede que la pivot no esté viendo las nuevas filas. Puedes editar el origen: Analizar > Cambiar origen de datos, y seleccionar el nuevo rango (mejor convertir a tabla así se adapta sola). Si era tabla estructurada, solo actualiza y tomará las filas nuevas.

Las tablas dinámicas pueden hacer cálculos más avanzados también: campos calculados, agrupar fechas en años/trimestres, agrupar números en rangos, etc. Como ejemplo sencillo, si tu tabla de datos tiene fecha, Excel a veces automáticamente agrupa por año y mes en la pivot (depende de opciones); si no, puedes click derecho en una fecha dentro de la pivot > "Agrupar" > elegir agrupar por meses, años, etc., y Excel creará campos de agrupación.

Importante: La tabla dinámica no es volátil ante cambios de estructura en datos. Si renombraste encabezados de la tabla original o quitaste columnas, la pivot puede perder esos campos.

Una tabla dinámica bien utilizada te ahorra crear decenas de fórmulas manuales (SUMAR.SI, CONTAR.SI, etc.) para obtener resúmenes. Además permite exploración rápida: supongamos que tu jefe te pide "dame totales por región", lo haces en segundos, luego dice "ahora por producto dentro de cada región", arrastras, listo; "ahora, de ese producto X, dime por mes" – podrías agregar Mes como filtro o filas, etc.

Ya con esta herramienta, estás equipado para analizar datos como un profesional. Capítulo 10: Macros y Automatización – Introducción a VBA en Excel

Llegamos a un nivel avanzado de Excel: las macros. Una macro es básicamente un programa (o secuencia de comandos) que se ejecuta dentro de Excel para automatizar tareas repetitivas o extender las capacidades de la hoja de cálculo. En Excel, las macros se escriben en el lenguaje VBA (Visual Basic for Applications), que es una versión de Visual Basic integrada en Office. Como programador, esta sección te resultará interesante porque podrás aplicar conceptos de código dentro de Excel. Veremos cómo grabar una macro, cómo acceder al editor de VBA para ver o escribir código, y haremos un ejemplo sencillo. También hablaremos de las consideraciones de seguridad y de cómo ejecutar macros en tus hojas.

Temas y objetivos:

AsuntosObjetivos

Editor de Visual Basic (VBA)Conocer el entorno de programación de Excel: cómo abrir el editor, encontrar los módulos, editar o escribir código.
Escribir y ejecutar macros sencillas (ejemplo)Ver un ejemplo de macro en VBA, entender su estructura básica (procedimientos Sub) y cómo ejecutarla (botón, atajo o automático)[12].

¿Qué es una macro y para qué sirve?

Como definimos, una macro es una acción o conjunto de acciones automatizadas. Puede ser tan simple como dar formato a un rango, o tan compleja como recorrer miles de filas tomando decisiones y generando reportes. Si hay tareas en Excel que haces frecuentemente de forma manual (por ejemplo, importar datos, limpiarlos, aplicar filtros, generar un informe con gráficos), con macros puedes hacer que un solo botón realice todo eso por ti en segundos[12]. Es como "grabar tus pasos y reproducirlos".

Grabadora de macros – tu primera macro sin escribir código

Excel ofrece una forma de crear macros sin saber programar: grabar una macro. Lo que hace es registrar cada clic y pulsación de teclado que hagas, traducirlo a código VBA, y guardarlo para que lo puedas ejecutar luego.

Pasos para grabar una macro:

Asegúrate de tener visible la pestaña Desarrollador (Developer) en la cinta. Por defecto está oculta. Para mostrarla: Archivo > Opciones > Personalizar cinta de opciones > marca "Programador" o "Desarrollador". Una vez habilitada, ve a esa pestaña.

En la pestaña Desarrollador, haz clic en Grabar macro (icono de círculo rojo si lo ves como rec).

Aparece un diálogo para nombrar la macro. Ponle un nombre sin espacios, por ejemplo FormatearTabla. Puedes asignarle una tecla de método abreviado (opcional, ten cuidado de no sobreescribir atajos existentes; si pones Ctrl+Shift+F, esa macro correrá con ese atajo). Y elige almacenar la macro en "Este libro" (por defecto). Pon también una descripción si deseas.

Pulsa Aceptar. Desde este momento, Excel está grabando. Realiza las acciones que quieras automatizar: por ejemplo, selecciona un rango, aplica un relleno de color, ordena una columna, inserta una forma, cualquier cosa soportada.

Cuando termines la secuencia de pasos, vuelve a Desarrollador y haz clic en Detener grabación.

¡Felicidades! Acabas de crear una macro. Ahora, ¿cómo la usamos? Podrías asociarla a un botón en la hoja (Insertar > Formas > dibuja un botón o forma, luego clic derecho Asignar Macro > eliges la macro). O ejecutarla directamente desde la lista de macros.

Para probar: Ve a Desarrollador > Macros, selecciona "FormatearTabla" (o el nombre que diste) y clic en Ejecutar. Excel repetirá las acciones grabadas. Si estabas en otro lugar o con otros datos similares, verás esos pasos replicados.

Editor de Visual Basic (VBE)

Detrás de la grabación, Excel creó código en VBA. Para verlo (y editarlo o escribir el tuyo propio), necesitas abrir el Editor de Visual Basic.

Ve a la pestaña Desarrollador, clic en Visual Basic (o presiona Alt + F11, atajo directo).

Se abrirá una ventana aparte, el editor. A la izquierda verás el proyecto de VBA con tu libro (Book1.xlsm, por ejemplo). Dentro, en Módulos debería estar Módulo1 (si grabaste macros, Excel las suele colocar en Módulos estándar).

Haz doble clic en Módulo1. A la derecha verás el código.

Algo así podría aparecer, por ejemplo:

Sub FormatearTabla()
'
' FormatearTabla Macro
' Aplica formato de tabla y ordena los datos
'
' Atajo de teclado: Ctrl+Mayús+F
'
Range("A1:C10").Select
Selection.Sort Key1:=Range("A1"), Order1:=xlAscending, Header:=xlYes
With Selection.Interior
.Pattern = xlSolid
.Color = 65535
End With
End Sub

Este es el código que la grabadora generó según tus acciones. Cada macro es un procedimiento Sub (subrutina) con un nombre (sin espacios). Las líneas que comienzan con ' son comentarios (la grabadora pone detalles como atajo, descripción).

Podemos observar que seleccionó un rango, luego lo ordenó por la primera columna, luego aplicó un color de relleno amarillo (65535 es vbYellow).

Como programador, verás que se puede limpiar o optimizar mucho (la grabadora tiende a grabar selecciones explícitas; a menudo se pueden escribir macros sin seleccionar objetos, manipulando directamente). Pero por ahora está bien.

Puedes editar este código directamente. Por ejemplo, podrías generalizarlo: en vez de Range("A1:C10"), usar ActiveSheet.UsedRange para formatear todo el rango usado, etc. Pero entraremos en simple.

Escribir una macro manualmente (ejemplo):

Vamos a hacer una mini macro de ejemplo: Un procedimiento que toma la celda activa y la pinta de rojo con texto en negrita "¡Atención!". Así practicamos escribir y ejecutar.

En el editor, Insertar > Módulo (si no tienes ya uno). Escribe:

Sub MarcarAtencion()
' Esta macro marca la celda activa con un formato de atención
With ActiveCell
.Value = "¡ATENCIÓN!"
.Font.Bold = True
.Interior.Color = vbRed
.Font.Color = vbWhite
End With
End Sub

Cada línea que comienza con . se aplica al objeto ActiveCell debido al bloque With. Cambiamos el valor de la celda a "¡ATENCIÓN!", ponemos la fuente en negrita, fondo rojo (vbRed es una constante de VB para rojo), texto blanco.

Cierra o minimiza el editor (los cambios se guardan automáticamente en el proyecto de VBA del libro). Ahora de vuelta en Excel, ve a Desarrollador > Macros, allí verás "MarcarAtencion". Pruébala: selecciona cualquier celda, ejecuta la macro. La celda debería ahora decir ¡ATENCIÓN! en blanco sobre rojo.

¡Has escrito tu primera macro en VBA! 🎉

Ejecutar macros: botones, atajos, eventos

Como mencioné, puedes asignar macros a botones o formas. Por ejemplo, Inserta una forma (un rectángulo), escribe encima "Marcar atención". Luego clic derecho esa forma > Asignar macro > elige MarcarAtencion. Ahora cuando hagas clic en el botón en la hoja, correrá la macro.

También macros se pueden asignar a eventos (por ejemplo, al abrir el libro, al cambiar una hoja, etc., pero eso es ya más avanzado y se hace escribiendo código en los módulos de eventos de Workbook o Worksheet).

Atajos de teclado se asignan al grabar o puedes cambiarlo en la lista de macros (seleccionas, Opciones, y ahí pones atajo).

Seguridad de macros y formato de archivo .xlsm

Las macros son esencialmente código que puede hacer casi cualquier cosa en tu sistema, así que Excel las trata con precaución:

Un libro con macros debe guardarse en formato .xlsm (Excel Macro-Enabled Workbook). Si intentas guardar en .xlsx, te advertirá que no puede guardar código ahí.

Al abrir un archivo con macros, dependiendo de tu configuración de seguridad, Excel puede deshabilitarlas automáticamente y mostrar una alerta (la "barra amarilla") diciendo "Las macros han sido deshabilitadas". Deberás hacer clic en "Habilitar contenido" si confías en el origen. Nunca habilites macros de un archivo de orígen desconocido o sospechoso, pueden contener código malicioso.

Es buena práctica firmar digitalmente macros si las compartes en un entorno corporativo, o colocarlas en ubicaciones de confianza definidas en Excel (Centro de confianza).

Puedes ajustar la configuración en Archivo > Opciones > Centro de confianza > Configuración de macros (por ejemplo, habilitar todas macros - no recomendado, o deshabilitarlas con notificación - la opción por defecto, etc.). Para desarrollo personal suele estar bien la opción con notificación.

Integración con Java y Python (visita rápida)

Aunque Excel por sí solo usa VBA para macros, tú preguntaste sobre integración con Java y conoces algo de Python.

VBA con otros lenguajes: VBA puede hacer llamadas a librerías COM o DLLs externas, incluso ejecutar comandos de sistema, pero eso es muy avanzado. Directamente Java no se ejecuta dentro de Excel.

Automatizar Excel desde Java: Existe la posibilidad de controlar Excel desde un programa Java usando bibliotecas. Por ejemplo, Apache POI es una librería Java que te permite leer y escribir archivos Excel (Excel no necesita estar abierto siquiera). Con POI puedes generar un .xlsx, editar celdas, fórmulas, etc., desde código Java[13]. También hay librerías para manipular Excel vía COM como JACOB (Java-COM Bridge) si quisieras controlar una instancia de Excel en Windows.

Python en Excel: Esta es una novedad reciente. Microsoft ha integrado la ejecución de Python dentro de Excel 365[14]. Puedes usar la función =PY() en una celda para ejecutar código Python utilizando librerías de análisis de datos (pandas, matplotlib, etc.) y devolver resultados a Excel[15]. Por ejemplo, podrías poner en A1: =PY("import math\nresult = math.sqrt(16)\n", "result") y Excel te devolvería 4 en la celda (esto indica: ejecuta código Python que calcula sqrt(16) y devuelve la variable result). Esta característica requiere tener la versión actualizada de 365 y posiblemente habilitar el complemento Python (no disponible offline).

Power Query / Power BI: Si tu objetivo es integrarte con bases de datos o hacer automatizaciones de ETL (extract-transform-load), la herramienta Power Query (en Datos > Obtener y transformar) permite conectar Excel a fuentes externas, transformar datos con una interfaz o lenguaje M (propio de Power Query), y actualizar con un clic. No es Java ni Python, pero es otra vía de automatización sin macros.

JavaScript para Office: Existe una API de Office Scripts (TypeScript/JavaScript) especialmente en Excel Online para automatizar tareas con un modelo de objeto moderno, pero en la aplicación de escritorio de Excel, la vía principal sigue siendo VBA (por ahora).

Ejemplo de integración: leer Excel desde Java con POI

Dado tu perfil Java, te interesará saber que con Apache POI podrías, por ejemplo, abrir un workbook, iterar filas y columnas, modificar valores y salvar. Un snippet conceptual en Java:

Workbook wb = new XSSFWorkbook(new FileInputStream("datos.xlsx"));
Sheet hoja = wb.getSheetAt(0);
for (Row fila : hoja) {
Cell celda = fila.getCell(0);
if (celda != null && celda.getCellType() == CellType.STRING) {
System.out.println(celda.getStringCellValue());
}
}
// ... (modificar celdas o crear nuevas) ...
FileOutputStream out = new FileOutputStream("salida.xlsx");
wb.write(out);
out.close();
wb.close();

Esto muestra lectura secuencial, pero POI tiene muchas capacidades (crear estilos, fórmulas, etc.)[13]. Para más información, consulta la documentación de POI o tutoriales específicos de Java-Excel. El punto clave es que Excel como aplicación no es necesario; POI maneja el archivo .xlsx directamente a nivel de estructura OpenXML.

Conclusión sobre macros

Las macros y la integración de código abren un mundo de posibilidades: desde automatizar formateos y cálculos repetitivos en libros personales, hasta desarrollar aplicaciones completas de Excel con formularios, botones, y lógica de negocio. Siempre ten presente escribir macros limpias, comentadas, y evitar acciones innecesarias (como .Select repetitivos) para que tu código sea eficiente.

Además, considera la mantenibilidad: si un proyecto de Excel con macros crece mucho, quizás sea mejor migrar a una solución más robusta (por ejemplo, usar Python + Excel para separar la lógica, o una base de datos y reporting). Pero saber macros te da una herramienta de productividad increíble para resolver muchos problemas rápidamente dentro de Excel mismo.
