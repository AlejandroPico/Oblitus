Curso organizado en capitulos para aprender Excel desde la base hasta un uso avanzado orientado a trabajo real, analisis y productividad.

Capitulo 1: Introduccion a Excel y conceptos basicos

Excel es una hoja de calculo que permite organizar datos en celdas, hacer calculos, crear tablas, analizar informacion y generar graficos. Un archivo de Excel se llama libro; cada libro contiene hojas; cada hoja esta formada por filas, columnas y celdas.

Capitulo 2: Crear libros, introducir datos y guardar

El primer flujo de trabajo consiste en crear un libro en blanco, escribir datos, distinguir texto, numeros y fechas, guardar el archivo y volver a abrirlo. La navegacion por celdas, rangos, filas y columnas es la base de todo uso posterior.

Capitulo 3: Formato y presentacion

El formato ayuda a convertir una tabla bruta en un documento legible. Incluye fuentes, bordes, colores, alineacion, formatos numericos, estilos de tabla, ajuste de texto y ancho de columnas.

Capitulo 4: Formulas esenciales

Las formulas permiten calcular resultados. Las mas basicas son suma, resta, multiplicacion y division. Despues aparecen funciones como SUMA, PROMEDIO, MIN, MAX, CONTAR y SI.

Capitulo 5: Referencias relativas, absolutas y mixtas

Una referencia relativa cambia al copiar una formula; una referencia absoluta se mantiene fija mediante el simbolo de dolar; una referencia mixta bloquea solo fila o columna. Dominar esto evita gran parte de los errores habituales.

Capitulo 6: Ordenar, filtrar y convertir datos en tablas

Las tablas de Excel permiten ordenar, filtrar, aplicar formato automatico y trabajar con rangos dinamicos. Son recomendables para cualquier conjunto de datos que vaya a crecer.

Capitulo 7: Funciones logicas y busquedas

SI, Y, O, SI.CONJUNTO, BUSCARX, BUSCARV, INDICE y COINCIDIR permiten crear hojas inteligentes que responden a condiciones y buscan datos en tablas relacionadas.

Capitulo 8: Graficos y visualizacion

Los graficos convierten datos en informacion visual. Es importante elegir el tipo correcto: columnas para comparaciones, lineas para evolucion temporal, circulares solo para proporciones simples y dispersion para relaciones entre variables.

Capitulo 9: Tablas dinamicas

Las tablas dinamicas resumen grandes cantidades de datos por categorias, fechas, importes y filtros. Son una de las herramientas mas potentes para analisis rapido sin programar.

Capitulo 10: Limpieza y preparacion de datos

Antes de analizar hay que limpiar: quitar duplicados, normalizar textos, corregir fechas, separar columnas, unir campos y revisar errores. La calidad de los datos condiciona la calidad del resultado.

Capitulo 11: Power Query y automatizacion de importaciones

Power Query permite importar, transformar y refrescar datos desde archivos, carpetas, bases de datos y servicios externos. Es clave para automatizar tareas repetitivas.

Capitulo 12: Macros y VBA

Las macros automatizan acciones. VBA permite programar procedimientos mas avanzados, crear botones, recorrer rangos y construir pequenas herramientas internas.

Capitulo 13: Buenas practicas profesionales

  • Separar datos, calculos y presentacion.
  • Evitar formulas innecesariamente largas.
  • Nombrar hojas y tablas con claridad.
  • Documentar supuestos y fuentes.
  • Proteger celdas criticas cuando el archivo lo usen terceros.

Proyecto final

El proyecto recomendado es un panel de control de ventas con datos importados, limpieza con Power Query, tabla dinamica, graficos, segmentadores y una hoja de resumen ejecutivo.
