# Artículo Oblitus: La anatomía política de España

Estructura generada para `articulos/anatomia-politica-espana/`.

## Archivos incluidos

- `index.html`: fragmento HTML con frontmatter YAML, sin `doctype`, `html`, `head` ni `body`.
- `assets/imagenes/`: portada y figuras editoriales SVG propias del artículo.
- `assets/js/congreso-escanos.js`: widget JavaScript aislado para visualizar la evolución agrupada del reparto de escaños.
- `assets/datos/congreso-escanos.json`: datos agrupados usados por el widget y conservados como recurso reutilizable.

## Notas editoriales

- El documento original se ha preservado en su extensión y estructura intelectual, reconvirtiendo la bibliografía en una lista ordenada con enlaces externos seguros.
- Las citas numéricas del texto se han convertido en superíndices enlazados a la sección de fuentes.
- El widget electoral utiliza agrupaciones editoriales para evitar una visualización ilegible con decenas de siglas históricas. No sustituye a una tabla electoral oficial desagregada por candidatura.
- Las fuentes originales incluyen materiales de naturaleza desigual —BOE, trabajos académicos, prensa, Wikipedia y YouTube—. Para una publicación académica estricta, convendría reforzar las partes más sensibles con fuentes primarias y bibliografía especializada.
