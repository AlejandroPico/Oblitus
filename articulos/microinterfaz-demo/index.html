Demostración de artículo con microinterfaz JavaScript

Ejemplo mínimo de una pieza editorial con control interactivo integrado.

7 jun 2026 · 4 min · Interactivo

Este artículo demuestra el formato que se utilizará cuando una pieza necesite un pequeño componente interactivo. La web sigue siendo estática, pero cada carpeta de artículo puede incorporar su propio JavaScript, sus imágenes, sus datos y sus audios.

Widget demostrativo

En un artículo real, este bloque podría representar una simulación, una tabla dinámica, una cronología, una comparativa de datos o un gráfico explicativo.

Uso previsto

La carpeta del artículo puede guardar recursos aislados para evitar mezclar lógica de distintas publicaciones. La portada solo necesita saber el título, resumen, etiquetas y ruta del artículo mediante posts.json.
