El descenso del gradiente es una de las ideas centrales del aprendizaje automatico moderno: ajustar parametros para reducir una funcion de perdida.

1. Idea central

Un modelo contiene parametros. Una funcion de perdida mide cuanto se equivoca. El descenso del gradiente actualiza esos parametros en la direccion que reduce la perdida con mayor rapidez local.

Regla esencial: parametro siguiente = parametro actual - tasa de aprendizaje por gradiente.

El parametro representa lo que el modelo puede ajustar. La perdida representa el error. El gradiente indica hacia donde aumenta mas la perdida. Como queremos bajar, avanzamos en la direccion contraria.

2. Intuicion geometrica

Imagina que estas en una montana con niebla. No ves todo el paisaje, pero puedes sentir la pendiente bajo tus pies. Si das pasos en la direccion de mayor bajada, repites la medicion y reajustas el rumbo, acabaras acercandote a un valle.

Gradiente

Es un vector de derivadas parciales. Indica como cambia la perdida cuando modificamos cada parametro.

Tasa de aprendizaje

Controla el tamano del paso. Una tasa muy baja hace que el aprendizaje sea lento; una demasiado alta puede provocar oscilaciones o divergencia.

3. Algoritmo paso a paso

  1. Inicializar parametros.
  2. Calcular la prediccion.
  3. Medir la perdida.
  4. Calcular el gradiente.
  5. Actualizar parametros.
  6. Repetir hasta converger.

4. Laboratorio minimo

Pulsa el boton para ver la trayectoria.

5. Regresion lineal

En regresion lineal buscamos una recta que aproxime datos. El descenso del gradiente ajusta pendiente e intercepto para minimizar el error cuadratico medio.

6. Variantes

  • Batch gradient descent: usa todos los datos en cada paso.
  • Stochastic gradient descent: usa ejemplos individuales y anade ruido util.
  • Mini-batch: equilibrio entre estabilidad y rendimiento.
  • Momentum: acumula direccion previa para atravesar zonas planas o ruidosas.

7. Diagnostico

Si la perdida baja lentamente, la tasa puede ser pequena. Si explota o oscila, puede ser demasiado grande. Si se estanca, puede haber mala escala de variables, mesetas o un problema de arquitectura.
