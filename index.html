Línea editorial

Investigación, divulgación y archivo crítico

La web está pensada como una biblioteca de artículos extensos: ciencia, tecnología, historia, geopolítica del conocimiento, inteligencia artificial y cultura crítica. El objetivo es publicar piezas densas, con estructura documental, pero legibles como una revista digital.

Publicación

Carpeta articulos/

Los documentos fuente se guardan en articulos/. Al subir cambios a GitHub, una acción genera el índice que alimenta la portada.

Formatos

Word, PDF, Markdown y HTML

El flujo recomendado es escribir en Word y publicar como .docx. Para artículos con JavaScript, se usa una carpeta con HTML y recursos propios.
